# Claude Switcher 安装指南

本指南为开发者和最终用户提供了 Claude Switcher 扩展的安装说明。

---

## 最终用户（推荐）

这种方法是安装扩展最简单的方式，无需直接处理源代码。

### 步骤 1：创建 ZIP 文件（针对项目所有者）

在您分发扩展之前，您需要将其打包成一个 `.zip` 文件。

1.  导航到您的项目目录 (`claude_switcher`)。
2.  选择该目录内的**所有文件和文件夹**（例如 `manifest.json`、`popup.html`、`assets/` 等）。
3.  **重要提示**：请勿选择父级 `claude_switcher` 文件夹本身。您必须选择它*内部*的内容。
4.  右键单击选定的文件，选择“压缩”或“发送到 > 压缩(zipped)文件夹”。
5.  将生成的文件命名为 `claude_switcher.zip`。这就是您将与用户分享的文件。

### 步骤 2：安装 ZIP 文件（针对最终用户）

1.  打开 Google Chrome 浏览器。
2.  在地址栏输入 `chrome://extensions` 并按回车键，进入扩展程序管理页面。
3.  在该页面的右上角，打开**开发者模式**开关。手动安装扩展需要此模式。
4.  现在，只需将您收到的 `claude_switcher.zip` 文件直接**拖放到 `chrome://extensions` 页面**上。
5.  扩展程序将被安装，并应出现在您的扩展程序列表中。如果您愿意，安装完成后可以关闭开发者模式。

---

## 开发者

这种方法适用于希望直接从源代码运行扩展的开发者。

1.  打开 Google Chrome 浏览器。
2.  在地址栏输入 `chrome://extensions` 并按回车键，进入扩展程序管理页面。
3.  在该页面的右上角，打开**开发者模式**开关。
4.  点击左上角出现的**加载已解压的扩展程序**按钮。
5.  在文件选择对话框中，导航并选择您的 `claude_switcher` 项目目录（即包含 `manifest.json` 的文件夹）。
6.  扩展程序将立即加载。如果您对代码进行了更改，需要点击此页面中扩展程序卡片上的“重新加载”图标才能看到更改生效。
